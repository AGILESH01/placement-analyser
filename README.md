# AI-Driven Placement Impact Analyzer & Employment Forecasting System

## Overview

This project is a web-based analytics system developed to analyze placement data from educational institutions and identify trends in employability, hiring patterns, and the impact of Artificial Intelligence on placements.

The system allows users to upload placement datasets in PDF and CSV formats, automatically extract information, perform statistical analysis, apply machine learning models, and generate future placement forecasts.

The main objective of this project is to help institutions, placement cells, and students understand placement trends and make data-driven decisions.

---

## Problem Statement

Placement data is often stored in reports and spreadsheets, making it difficult to analyze long-term trends and extract meaningful insights.

With the rapid growth of Artificial Intelligence and automation, hiring patterns are changing significantly. Educational institutions need a way to understand these changes and prepare students for future job markets.

This project addresses that challenge by combining data analytics, statistics, and machine learning into a single platform.

## Machine Learning Approach

The project uses multiple machine learning techniques to analyze placement data.

### Linear Regression
Used for predicting future placement percentages and salary trends.

### Logistic Regression
Used to classify placement performance into high and low categories.

### Random Forest
Used to identify important factors influencing placement outcomes.

### K-Means Clustering
Used to group institutions with similar placement characteristics.
